# Neura-Self: Absolute Stealth 3-Account Setup Guide

This guide covers the specialized configuration for running **3 separate accounts** with **Maximum Behavioral Stealth** and **Infinite Multi-Account Isolation** on Termux or Desktop.

## 1. Prerequisites

Ensure you have the latest environment repairs:
- Run `python3 neura_setup.py` and select **`[3] Repair Environment`**.
- This ensures `onnxruntime` and `rich` are correctly configured.

## 2. Setting Up Your 3-Account Stealth Rig

Once inside the Setup Wizard (`neura_setup.py`), follow this sequence:

### Step A: Apply Modern Stealth Profile
Select **`[4] Apply Absolute Stealth (v2)`**. 
- This migrates the 32-phase humanization overhaul into your active `settings.json`.
- This activates Burst Typing, Fatigue Simulation, and Post-Captcha Stress Mode.

### Step B: Add Your 3 Accounts
Select **`[2] Manage Accounts`** and select **`[1] Add Account`**. **Repeat this 3 times** for each account:

1.  **Account 1 (Main)**:
    - **Name**: `Main`
    - **Token**: [Enter Your Token]
    - **Channel IDs**: [Enter Channel ID]
2.  **Account 2 (Alt 1)**:
    - **Name**: `Alt 1`
    - **Token**: [Enter Your Token]
    - **Channel IDs**: [Enter Channel ID]
3.  **Account 3 (Alt 2)**:
    - **Name**: `Alt 2`
    - **Token**: [Enter Your Token]
    - **Channel IDs**: [Enter Channel ID]

### Step C: Launch
Select **`[1] Start NeuraSelf`**.
